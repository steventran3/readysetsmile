# Ready Set Smile Booth

One-page website for [readysetsmilebooth.com](https://readysetsmilebooth.com), built with plain HTML, CSS, and JavaScript for GitHub Pages.

## Main files

- `index.html` — site structure and content
- `styles.css` — visual design and responsive layouts
- `script.js` — navigation, scroll effects, package selection, and inquiry email builder
- `CNAME` — keeps the custom domain connected to GitHub Pages
- `www/img/` — event and booth photography

## Inquiry form

The inquiry form prepares an email to `info@readysetsmilebooth.com` using the visitor's default email application. This works on GitHub Pages without a server or third-party form account.

## Publishing

GitHub Pages serves the `master` branch. Merge an approved redesign branch into `master` to publish it to the custom domain.
